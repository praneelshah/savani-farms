## Savani Farms – Mango Site

This is a Vite + React + TypeScript single-page site inspired by Savani Farms, showcasing premium Indian mangoes and related content.

### Getting started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available on `http://localhost:5173` (or the port Vite prints in the terminal).

### Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Build and preview

```sh
npm run build
npm run preview
```
